Using Bootstrap Classes for Images

This page demonstrates how to use Bootstrap classes to style and manage images effectively.

Responsive Images

In Bootstrap, you can make an image responsive using the .img-fluid class, which scales the image to fit the width of its parent container while maintaining the aspect ratio.

This is a responsive image that adjusts its size based on the screen width.

Rounded Images

Use the .rounded class to add rounded corners to your images.

This is an image with rounded corners.

Circle Images

Use the .rounded-circle class to make your image a perfect circle.

This is a circular image.

Thumbnail Images

Use the .img-thumbnail class to add a border around the image and a small padding.

This is a thumbnail image with a border and padding.

Image with Caption

Bootstrap allows you to add captions to images using the .figure and .figure-caption classes.

This image is displayed with a caption using Bootstrap's figure classes.

Inline Text Elements

Bootstrap also provides a few inline text elements for styling parts of the text:

  • .text-muted for muted or less emphasized text.
  • .text-decoration-underline for underlined text.
  • .text-decoration-line-through for strikethrough text.
  • .text-nowrap to prevent text wrapping.
